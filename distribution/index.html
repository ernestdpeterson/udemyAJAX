Creat Your own JSON

The only code you need to get an ajax response is:
var newJason = "https://api.myjson.com/bins/1dango"; // the endpoint definition
var xhr = new XMLHttpRequest(); // the new request object
xhr.open('GET', newJason); // the method of request
xhr.send(); // sending the request
xhr.onreadystatechange = function() {/*Check for readyState 4 and status 200*/};
This will return a string of the entire JSON.
To get this string into a usable format you:
JSON.parse(xhr.response)
